## v0.3 — 2025-05-08

### 本版本說明
角色架構確立、後勤功能大幅強化、新增 Super Admin 上帝模式。
產品資料從真實追貨表.xlsx完整提取（BNE 112項 / GSE 105項 / SONY 132項，共 349 項）。

### 角色與密碼

| 角色 | 入口 | 密碼 |
|------|------|------|
| 經銷商 | 專屬連結（dealer.html?code=XX2025） | 無需密碼 |
| 業務 AGM | 登入頁下拉選名 | 無需密碼 |
| 後勤統計 | 登入頁 | `agm2025` |
| PM 管理 | 登入頁 | `pm2025` |
| Super Admin | 登入頁 | `super2025` |

### 更新內容

#### 經銷商填單（dealer.html）
- 三層分類：代理商大 Tab（BNE / GSE / SONY）→ 平台或類型子 Tab → 品項列表
- BNE 子 Tab 按平台分（NS / NS2 / PS5 / PS4）
- GSE 子 Tab 按類型分（軟體 / 女性向 / 誠翼 / OLP）
- SONY 子 Tab 按類型分（主機 / 周邊 / 軟體 / PS4 HITS / VR / SEGA）
- 顯示國際條碼（格式：`國條:4713014XXXXXX`）
- 到貨日橘色標籤（📅 到貨日：XXXX），PM 回填後即時顯示
- 分配出貨標籤（⚠ 分配出貨）
- 搜尋列：支援品名或國際條碼即時過濾
- 收單時間提示：後勤設定起始/截止時間後，填單頁頂部自動顯示

#### 後勤統計（logistics.html）
- MOQ 警示區：自動列出不足品項，顯示已下單通路（綠）vs 未下單通路（紅）
- 通知 Sales 彈窗：
  - 全部通知：一次產生所有業務的完整催單訊息
  - 按業務分頁：各業務獨立一份，只列自己轄區缺口
  - 一鍵複製，可直接貼到 Line / Email
- 通路填單狀況強化：
  - 可按業務篩選、可按狀態篩選（已送出 / 填寫中 / 未填單）
  - 每通路可一鍵複製專屬連結
- 統計卡片可點擊快速篩選（MOQ 不足、未填單通路）
- 後勤報表 CSV 匯出（含各通路分布、向原廠下單量、剩餘可用）
- 業務報表 CSV 匯出（含各通路明細、MOQ 缺口警示）
- Tab：統計總覽 / 通路連結 / 週期設定（含起始/截止時間）

#### PM 管理（pm.html）
- Tab 順序調整：商品管理（優先）→ MOQ 回填
- 商品管理顯示 AGM P/N + 國際條碼
- 多層篩選（代理商 / 平台 / 類型 / 關鍵字搜尋）
- 貨物狀態下拉選單（對應前台顯示）
- 移除通路連結和週期設定 Tab（改由後勤統計管理）

#### Super Admin（superadmin.html）— 新增
共 9 個功能 Tab：

| Tab | 功能 |
|-----|------|
| 📊 總覽 | 全品項即時彙整 + MOQ 向原廠下單回填 |
| 📦 商品管理 | 批次 CSV 匯入 + 手動新增 + 多層篩選 |
| 📋 MOQ 回填 | 狀態/到貨日/向原廠下單量維護 |
| 🏪 通路管理 | 新增/刪除通路、調整負責業務 |
| 💼 業務管理 | 業務名單 + 轄區訂單彙整 |
| 📑 報表匯出 | 後勤報表 / 業務報表 / 經銷商報表（三種） |
| 🔗 通路連結 | 複製 / 匯出 CSV |
| 📅 週期設定 | 開關收單視窗 + 起始/截止時間 |
| ⚙️ 系統設定 | 密碼說明、Firebase 資訊、危險操作 |

批次匯入 CSV 格式（14 欄，第一列為標題跳過）：
```
代理商,平台,類型,AGBS P/N,國際條碼,中文品名,規格,SRP,出貨價(未稅),優惠出貨價,MOQ,到貨日,狀態,分配出貨(true/false)
```

#### 登入頁（index.html）
- 新增 Super Admin 角色按鈕
- 移除重複按鈕

## v0.2 — 2025-05-07

### 本版本說明
Firebase Realtime Database 整合完成，從 localStorage 完全遷移，支援多人即時同步。
UI 全面改為淺色主題，針對 40 歲以上使用者優化。

### 更新內容
- Firebase 整合：即時同步、300ms debounce 防抖
- 淺色主題 UI（#f0f2f5 背景、白色卡片）
- 字體放大（內文 0.92-1rem、標題 1.6rem）
- 收單視窗開關（Firebase 即時同步，關閉後自動鎖定前台）
- MOQ 缺口警示（紅字/橘字）
- 通路專屬連結產生與複製
- URL code 驗證（錯誤 code 顯示鎖定頁面）
- 匯出 CSV（含 BOM，支援 Excel 中文）

## v0.1 — 2025-05-07

### 本版本說明
平台初始建立，取代手動 Excel 追貨流程。

### 建立內容
- 專案架構確立（純靜態 HTML 5 頁面）
- 四個角色：經銷商 / 業務 AGM / 後勤統計 / PM 管理
- localStorage 版本資料層（db.js）
- 經銷商填單頁（dealer.html）：品項列表 + 數量輸入 + 送出確認
- 業務檢視頁（sales.html）：轄區通路訂單、MOQ 缺口
- 後勤統計頁（logistics.html）：全品項彙整、通路明細
- PM 管理頁（pm.html）：商品管理、MOQ 回填
- 登入頁（index.html）：角色選擇 + 密碼驗證
- Netlify 部署設定
